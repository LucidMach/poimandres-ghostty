# Poimandres-Ghostty

Unofficial [poimandres](https://github.com/z0al/poimandres-alacritty/) theme for [ghostty terminal](https://ghostty.org/)

## Installation

Copy the content of either poimandres-storm or poimandres to `~/.config/ghostty/config`.
